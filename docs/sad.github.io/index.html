SAD: Style-Aware Diffusion Adaptation for Few-shot Style Transfer Image Generation

Yilong Liu, Houwen Zheng, Shuojin Yang
Tsinghua University
Paper Code

SAD achieves efficient and robust image style transfer with only a few reference examples.

Abstract

Diffusion-based models have recently demonstrated remarkable success in image style transfer. However, existing methods struggle to balance style fidelity and content preservation when training data is scarce.

We present SAD (Style-Aware Diffusion), a parameter-efficient style transfer framework that leverages low-rank adaptation for few-shot learning. SAD extracts latent representations of both style and content via DDIM inversion, formulated as an ordinary differential equation, enabling precise disentanglement of stylistic and structural information.

To effectively align low-level style features with high-level content semantics, we introduce a hybrid strategy combining adaptive instance normalization and query-key-value injection within the diffusion process. Furthermore, SAD employs a parameter-efficient adaptation scheme that selectively optimizes attention layer weights, mitigating catastrophic forgetting and overfitting. Extensive experiments demonstrate that SAD significantly outperforms prior diffusion-based style transfer methods, achieving a 61.5% relative performance improvement over the plain model and surpassing the strong DreamBooth-LoRA baseline.

Method Overview

Overall Framework of SAD. In Stage I, we obtain coarse target image and corresponding intermediate latent representations. Stable Diffusion models (SDMs) are employed to generate initial content from target prompts, followed by selection of the optimal style reference based on the similarity of CLIP text embeddings. Subsequently, content and style images are encoded into latent space via DDIM inversion in the form of an ODE, with self-attention features ${Q, K, V}$ recorded across multiple timesteps. In Stage II, the fine target image is generated from refined latent features with an SDM. AdaIN is utilized to align the statistical properties of content and style within the latent representation, and queries seamlessly integrate content features with those from the final timestep, while keys and values are derived from the style reference. PEA, a proposed optimization method, is used to prevent overfitting and catastrophic forgetting.
